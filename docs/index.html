Installation

Download directly or build from source

Download (Recommended)

  1. Go to Releases and download the latest .zip
  2. Unzip and drag LilyAisland.app to Applications
  3. Launch, grant Accessibility permission when prompted
  4. The island appears at the top center of your screen

Build from Source

git clone https://github.com/CHOWLAN233/LilyAisland.git
cd LilyAisland
open LilyAisland.xcodeproj

Requires Xcode 16+, macOS 15.2+. Product > Archive or use xcodebuild.
