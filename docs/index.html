📥 安装方式

两种方式任选 — 直接下载或从源码构建

⬇️ 下载安装（推荐）

  1. 前往 Releases 页面 下载 .zip
  2. 解压并拖入「应用程序」文件夹
  3. 启动应用，根据提示授予「辅助功能」权限
  4. 灵动岛将出现在屏幕顶部中央 ✨

🛠️ 从源码构建

git clone https://github.com/CHOWLAN233/LilyAisland.git
cd LilyAisland
open LilyAisland.xcodeproj

需要 Xcode 16+，macOS 15.2+。Product > Archive 或使用 xcodebuild 命令行构建。
